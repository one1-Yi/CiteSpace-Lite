# 🌐 CiteSpace-Lite（毕设小项目）
### 轻量级文献计量分析与可视化系统  
基于 Flask + ECharts 实现的文献数据分析、统计与可视化工具。

---

## ✨ 系统功能特性
- 📊 **高产作者 TOP15 统计**（红色条形图）
- 📚 **文献来源期刊 TOP15 统计**（橙色条形图）
- 🔍 **高被引文献 TOP20 统计**（蓝色柱状图，可点击跳转详情）
- 📈 **多图表样式切换**：柱状图 / 环形图 / 雷达图
- 🎨 **统一深色主题**，界面美观、专业
- 🧮 **数据自动清洗**，过滤空值、Nan、无效字段
- 🖱️ **高级交互**：被引文献图表支持点击，跳转至文献详情
- 📱 **响应式布局**，窗口缩放不错乱、不重叠

---

## 🛠️ 技术栈
- **后端**：Python 3.x + Flask
- **前端**：HTML5 + CSS3 + JavaScript
- **可视化库**：ECharts 5.x
- **数据处理**：Pandas
- **文件格式**：CSV 文献数据
